# LearnLoop – Learning Platform

LearnLoop is a simple learning platform built using React. It provides an interface for users to explore learning resources and interact with educational content.

## Technologies Used

* React.js
* JavaScript
* HTML
* CSS

## Project Structure

* public → Static files
* src → React components and application logic

## How to Run the Project

Install dependencies:

npm install

Start the project:

npm start

The application will run at:
http://localhost:3000
